# Agent Instructions

This repository's agent instructions are maintained as a **portable instruction
set** under [`.agents/`](.agents/). This file is intentionally universal: the
same `AGENTS.md` is used across every repository, and all the actual rules live
in `.agents/`.

**Start with [`.agents/INDEX.md`](.agents/INDEX.md)**, which indexes every rule
and reference file, then read the files it points to before making any change.

| Area | Where |
|---|---|
| Index of everything | [`.agents/INDEX.md`](.agents/INDEX.md) |
| Behavioural rules (iron rules, execution) | [`.agents/rules/`](.agents/rules/) |
| Git & branching workflow | [`.agents/git/`](.agents/git/) |
| Repository architecture, templates, and this site's structure | [`.agents/architecture/`](.agents/architecture/) |
